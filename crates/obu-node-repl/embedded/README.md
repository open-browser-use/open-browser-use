# Embedded JavaScript Assets

- `kernel.js` is the open-browser-use Node VM kernel used by `obu-node-repl`.
- `meriyah.umd.min.js` is vendored from npm package `meriyah@7.0.0`.

Do not regenerate these files blindly; preserve the OBU contract details
(`display`, `__obuNativePipe`, OBU env vars, `exec_result.result`).
